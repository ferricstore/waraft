# Code of Conduct

This project expects professional, respectful collaboration.

## Expected behavior

- Be direct, technical, and constructive.
- Assume good intent, but be precise when identifying risks or mistakes.
- Keep discussions focused on the code, design, tests, operations, and users.
- Respect maintainers' decisions on scope and release timing.

## Unacceptable behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without consent.
- Repeatedly derailing technical discussions after maintainers ask to stop.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests;
and block accounts when needed to keep the project usable.

For sensitive concerns, contact the repository maintainers privately rather than
opening a public issue.
