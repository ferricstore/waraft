# Security Policy

## Supported versions

The current supported line is:

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a vulnerability

Please do not report vulnerabilities through public GitHub issues.

Preferred reporting path:

1. Use GitHub private vulnerability reporting for this repository, if enabled.
2. If that is unavailable, contact the repository owner/maintainers privately.
3. Include a minimal reproduction, affected version, expected impact, and any known mitigations.

We will acknowledge reports as soon as practical and coordinate fixes before
public disclosure when the issue is credible.

## Scope

This policy covers the `wa_raft` Erlang library in this repository.
FerricStore-specific storage/log adapters and NIFs are maintained in the
FerricStore repository and should be reported there.
